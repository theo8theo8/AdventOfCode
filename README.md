# AdventOfCode 2020/2021
My solutions to Advent of Code 2020, 2021 and 2022.
2020 solutions are made in Java,
2021 and 2022 solutions are made in Python

For 2020:
To run, simply use
```bash
javac Filename.java
```
and then
```bash
java Filename
```

For 2021 and 2022:
To run, use
```bash
python3 filename.py
```
